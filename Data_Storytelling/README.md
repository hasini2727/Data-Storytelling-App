# Data Storytelling - Titanic Survival Analysis

## Overview

This project analyzes the Titanic dataset.
It shows survival patterns using charts and simple observations.

## Features

* Dataset preview
* EDA
* Survival analysis
* Visualizations
* Conclusion

## How to Run

1. Install required libraries:
pip install streamlit pandas plotly numpy

2. Run the app:

python -m streamlit run app.py

## Dataset

Titanic Dataset
