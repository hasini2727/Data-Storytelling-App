# Smart Analytics Tool

## Overview

This project allows users to upload a CSV file and analyze it.
The tool generates statistics and charts automatically.

## Features

* CSV upload
* Dataset preview
* Missing value analysis
* Statistical summary
* Scatter plot
* Histogram
* Box plot

## How to Run

1. Install required libraries:

pip install streamlit pandas plotly numpy

2. Run the app:

python -m streamlit run app.py
